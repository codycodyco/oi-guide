## 算法竞赛

一场算法竞赛通常持续数小时，你需要编程解决一系列问题。这些问题不是公开的问题；

问题编写者和测试人员已经解决了这些问题，你需要在比赛的短时间内编程解决他们。

一般来说，算法竞赛中的每个问题都通过两步过程来解决：

1.  通过技巧和直觉提出合适的算法
2.  通过编程，将算法表现成代码

对于每一个问题，你都要将完成的代码提交给评测程序，评测程序会将许多问题输入到你的代码执行的程序中，并检查程序是否按照要求计算以及输出了正确的答案。对于每个问题，程序必须在规定的时间内计算完成（通常为2秒）和内存限制（通常是256m）。

请注意，算法竞赛和进行软件开发不一样，你只需要保证程序快速运行并输出正确答案即可，代码可读性在此时不太重要，勉勉强强能读懂即可。

举个不太恰当的例子，软件开发类似于开货车完成装货，拉货，卸货等任务，每一步都十分重要而且需要许多人合作完成，但算法竞赛类似于在赛场上开赛车，我们只需要尽可能快地到达终点即可

## 解决方案 - A+B Problem

读入两个数，并输出答案即可

<LanguageSection>

<CPPSection>

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a,b;
    cin >> a >> b;
    cout << a+b;
    return 0;
}
```

</CPPSection>

</PySection>

<JavaSection>

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int a = scanner.nextInt();
        int b = scanner.nextInt();
        System.out.println(a + b);
    }
}
```
