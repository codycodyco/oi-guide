# hello world与代码框架

## 代码框架

　　下面是一个最基础的代码框架。

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	//你的代码从这里开始写
}
```
　　我们的程序是从两个大括号之中开始，一行一行运行的。对于这些语句，我们可以认为他们是程序运行必不可少的框架，类似于一座大楼的承重墙。

　　下面的例子是对代码框架的具体解释，如果你看不懂，可以先跳过！
```cpp
#include<bits/stdc++.h>//头文件，相当于你的工具箱，很多代码工具都需要从这里取出
using namespace std;//展开命名空间，有的代码语句很长，用了它以后可以让代码变短
int main()//主函数，你的代码从主函数的第一行开始
{
	//你的代码从这里开始写
}
```

<Info title="为什么我的教练让我头文件写 #include<iostream> ?">
当我们编写头文件时，相当于从头文件的目录中准备了一些工具以供使用，当需要的工具变多后，需要写的头文件也会变多，你甚至可能需要写七八行头文件

然而，bits/stdc++叫做万能头文件，写下他相当于写下了几乎所有我们常用的头文件。在过去，一些OJ和比赛不支持万能头，但随着时间的推移，绝大多数OJ和比赛都已经支持万能头了,因此你完全可以放心使用它。
</Info>

　　对于初学者，你只需要知道代码框架是一个程序必不可少的内容即可。

## 输出一条语句

这是一个用于输出hello world的代码，你可以在自己的IDE上尝试运行这段代码。[如何在IDE上编译运行代码？]()
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	cout<<"hello world";
}
```
```
输出结果：
hello world!
```

　　除了之前编写的代码框架之外，我们还添加了一行 `cout`开头的代码,这是我们要学习的第一条语句。

　　`cout`是一条用于输出的语句，在cout中，我们可以直接输出以`" "`这样的**英文双引号**所包括的语句，例如：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	cout<<"hello c++！";
}
```
```
输出结果：
hello c++!
```
　　我们将双引号中的内容改为“hello c++！”，可以发现此时的输出结果已经变成了“hello c++!”，你还可以将其改为自己喜欢的句子！

　　我们注意到，cout这条语句之后还带有一个分号`;`，在c++中，分号代表一条语句的结束，类似于中文中的句号，分号必不可少，否则我们的代码将**无法编译运行！**

　　当我们使用cout进行输出时，也可以通过`<<`符号来进行连续的输出，例如：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	cout<<"hello"<<" "<<"world!"<<endl;
}
```
```
输出结果：
hello world!
```
　　通过连接符号`<<`，我们依然输出了`hello world`这段话，而最后的`endl`用于在cout中输出一个换行，当我们多写几条cout，就能看出它的作用，例如：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	cout<<"hello"<<" "<<"world!"<<endl;
	cout<<"hello"<<" "<<"c++!";
}
```
```
输出结果：
hello world!
hello c++!
```
　　由于我们的程序是按行运行的，所以此时程序先输出了`hello world!`和一个换行，又输出了`hello c++!`这句话。

　　如果你想单纯输出数字，那也是可以的，例如：
```cpp
	cout<<23;
```
<Info title="为什么这段代码没有代码框架?">
	代码框架太长了，所以在许多样例代码中会被省略，自己运行时需要加上代码框架。
</Info>

　　你甚至可以在输出中输出一个计算式！例如：
```cpp
	cout<<(123+123)/2*7-6;
```
请你在自己的电脑上将这些代码补上代码框架并尝试运行，看看会输出什么结果，然后再进入下一章节的学习。
